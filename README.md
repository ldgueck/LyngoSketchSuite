
### LyngoSketch Suite

I’m working on a.o. a family history project and haven't found the software that I need. So I built some.

#### What it is

**LyngoSketch** is a simple set of tools for research and writing. The name comes from "Lingo" (how people talk) and "Sketch" (a quick outline of an idea without extra clutter) and **Lynn** (Gaelic: from linne or linn, referring to a body of water or a deep pool below a waterfall, or, my preference, a forest lake), that's me, **Lynn**.

#### Why I built it

I needed software that could handle specific problems, like transcribing my grandmother's thick Volhynian high-low German accent—something the big commercial programs couldn't wrap their heads around. I also wanted my notes to be stored in plain files that can still be read in 50 years, even if this software is long gone.

#### What's in the box

*   **LyngoSketch WIKI:** My "digital brain" for linking notes together. It's flexible — I can run it **locally on my laptop or host it from a server**. It cleans up its own mess and keeps everything private.   

*   **LyngoSketch CAT:** A translation tool that uses **Gemini AI (via an API key)**. It doesn't just swap words; it lets me pick a "mood" and add context, so the translation sounds right for the story I'm telling.   

*   **LyngoSketch Transcribe:** A tool for turning old family tapes into text. Like CAT, it calls **Gemini AI with an API key** to handle rambling thoughts and tough accents that confuse standard apps.
It handles audio and OCR of images and PDFs.

*   **LyngoSketch TextMorph:** A utility convert rich text to **Markdown, HTML, LaTeX, or plain text**.
